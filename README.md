Codebreaker Client

## Summary

This project specifies an AI-assisted coding project problem (creating a console-mode client for the [Codebreaker Solitaire service](https://ddc-java.services/codebreaker-solitaire/) <https://ddc-java.services/codebreaker-solitaire/> to be completed in the [Deep Dive](https://deepdivecoding.com/) Java training programs.
